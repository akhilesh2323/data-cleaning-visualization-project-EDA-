# Exploratory Data Analysis (EDA) Project

## Objective
Analyze student performance data to identify trends, patterns, and relationships between study hours and scores.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis Performed
- Data Inspection
- Missing Value Analysis
- Duplicate Detection
- Statistical Summary
- Histogram
- Scatter Plot
- Box Plot
- Correlation Heatmap

## Key Findings
- Strong positive correlation between study hours and scores.
- Students studying more hours tend to achieve higher scores.
- Dataset contains minimal data quality issues.

## Conclusion
Exploratory Data Analysis helped uncover important relationships in the dataset and provided valuable insights for future predictive modeling.
